### Install notes for Zeyang Lin's MetaSearch Web Application ###

1. Install the following software/packages:
    *Apache httpd 2.2.22
    *Python 2.7.3
    *Django 1.4
    
2. Install "requests" module for Python, using 'requests-0.13.3.tar.gz' in this zip file.

3. Install "mod_wsgi" module for Apache, using 'mod_wsgi-3.3.tar.gz' in this zip file.

4. Unzip "Project" folder as well as all its contents into whichever folder you want to deploy this application.

5. Edit "settings.py" in 'Project/mysite/',
    change the path '/home/zeyang/MetaSearch/Project/static' in 'STATICFILES_DIRS' setting into 'your_path/Project/static' where 'your_path' is the folder into which you extract the 'Project' folder;
    change the path "/home/zeyang/MetaSearch/Project/templates" in 'TEMPLATE_DIRS' setting into 'your_path/Project/templates' where 'your_path' is the folder into which you extract the 'Project' folder.
    
6. Edit "httpd.conf" in 'your_apache_path/conf/' folder where 'your_apache_path' is the folder your Apache is installed,
    append the following lines at the end of the file:
    ######################################################
    LoadModule wsgi_module /usr/local/apache2/modules/mod_wsgi.so #change the path into the actual path of mod_wsgi.so

    Alias /static/ /home/zeyang/MetaSearch/Project/static/ #change the path into the actual path of /static/

    <Directory /home/zeyang/MetaSearch/Project/static/> #change the path into the actual path of /static/
    Order deny,allow
    Allow from all
    </Directory>

    WSGIScriptAlias / /home/zeyang/MetaSearch/Project/mysite/wsgi.py #change the path into the actual path of wsgi.py
    WSGIPythonPath /home/zeyang/MetaSearch/Project #change the path into the actual path of Project

    <Directory /home/zeyang/MetaSearch/Project/mysite> #change the path into the actual path of mysite
    <Files wsgi.py>
    Order deny,allow
    Allow from all
    </Files>
    </Directory>
    ########################################################
    
7. Launch Apache server, then enter the URL of the server in any browser.
